# Camilamboglia Studio — Landing Page

Landing page institucional para **Camilamboglia Studio**, agencia creativa especializada en branding, experiencias digitales y redes sociales.

## Propósito

Presentar los servicios del estudio a potenciales clientes en mercados hispanohablantes, mostrar trabajos anteriores y facilitar el contacto comercial.

## Tecnologías

- **Framework:** Next.js 16 con React 19
- **Estilos:** Tailwind CSS 4 con animaciones personalizadas
- **UI:** Radix UI + librería de componentes propia (shadcn-style)
- **Backend:** API route en Next.js con Nodemailer para envío de emails
- **Lenguaje:** TypeScript

## Secciones principales

| Sección | Descripción |
|---|---|
| Hero | Pantalla completa con marquee animado de servicios principales |
| About | Presentación del estudio y su filosofía |
| Services | Detalle de los servicios ofrecidos (Branding, Digital, Social Media) |
| Results | Métricas e historias de éxito |
| Content | Galería de trabajos y proyectos |
| Talent | Presentación del equipo |
| Contact | Formulario de contacto con validación y envío por email |

## Características destacadas

- Animaciones de scroll reveal en todas las secciones
- Formulario de contacto con campos: nombre, email, teléfono, empresa, rubro y necesidades
- Diseño responsive optimizado para mobile
- Paleta minimalista en tonos tierra (`#1a1a1a`, `#4a403a`, `#e4e2dd`)
